广而告之, 本站 RoadMap
主旨: 分享和交流使用 Emacs , 享受自由学习和折腾的乐趣.

  • p0 done@2015-03-28 按照两列罗列推荐文章和 Emacs 圈里的文章
  • p0 done@2015-03-28 支持 eww 访问
  • p1 ing 支持简单的登录系统
  • p1 ing eww 上的 minor-mode: emacsist-mode
  • p1 ing emacsist-mode 支持类 v2ex 的 append 和 comment(仅登录用户)
  • p1 ing emacsist-mode 支持一键推荐 emacs 文章(暂不支持自由发文讨论)
  • p2 emacsist-mode 定时检查有最新回复和更新; 针对访问时间以及是否查看分别标示
  • p3 支持个性定制频率等邮件订阅和讨论(邮件组), 最低一年一次邮件, 智能调整频率
  • p4 分中文和英文介绍 Emacs 圈的同学以及事迹, 罗列 blog 等
  • p5 支持分 tag(label@github), reddit 分子版块查看, 考虑设版块管理员机制
  • p5 支持5级的难度和5级的(质量)评分
  • p5 建立登录账户等级机制, 适当等级允许提问
  • p6 重点优化下网页访问, 包括首页 js/css 等的加载速度等
  • p7 整理下本站相关代码(in PHP CodeIgniter)并开源和邀请参与
  • p8 以 gif 动画和视频演示为切入点, 推荐 mode 的使用实践
  • p20 支持文章的评级等各种文章质量评估, 直接影响回复是否被顶起
  • p100 emacsist-mode 结合 org 或 outline , 根据标题等自动控制显示
  • p101 以 better practices/workflow 方式推荐和讨论使用实践组合
  • p102 类似 p101 的专题比如 php 开发方式推荐 实践组合
  • p200 本站一开始是 Hick 同学维护, 可行的话希望后边定好规则大家一起来运作
  • p222... A huge wave of ideas is coming
